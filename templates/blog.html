🚀 What is DataGPT?

DataGPT is an AI-powered tool that generates datasets based on user prompts. It helps in testing machine learning models and data analysis.

💡 Features

  • Generate datasets using natural language
  • Add missing values, noise, duplicates
  • Download datasets as CSV
  • Very helpful for students
